Networking & Security Interview Questions

Q1: What is the OSI model?

A1: The OSI model is a conceptual framework used to understand and implement networking protocols in seven layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.

Q2: What is the difference between TCP and UDP?

A2: TCP is a connection-oriented protocol that ensures reliable data transmission, while UDP is connectionless and doesn't guarantee delivery, making it faster but less reliable.

Q3: How do you secure a web application?

A3: Securing a web application involves using HTTPS, input validation, implementing firewalls, regular patching, and monitoring for vulnerabilities.
