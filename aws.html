AWS Interview Questions

Q1: What is AWS, and why is it used?

A1: AWS (Amazon Web Services) is a cloud computing platform that provides a wide range of services such as computing power, storage, and databases. It is used for hosting applications, data storage, and scaling resources.

Q2: What is an EC2 instance?

A2: An EC2 instance is a virtual server in AWS's Elastic Compute Cloud (EC2) service that allows you to run applications on the cloud.

Q3: How do you secure an S3 bucket?

A3: You can secure an S3 bucket by implementing bucket policies, enabling encryption, and ensuring proper IAM roles and permissions.
